Download a single file using ftp sockets.

URL syntax:
ftp://[<user>:<password>@]<host>/<url-path>

Examples:

./ftp ftp://ftp.up.pt/pub/mozilla/firefox/releases/19.0.2/linux-x86_64/ach/firefox-19.0.2.tar.bz2

./ftp ftp://anonymous:pass@ftp.up.pt/pub/CPAN/CPAN.html

./ftp ftp://ftp.up.pt/pub/CPAN/CPAN.html

./ftp ftp://ftp.up.pt/pub/robots.txt

./ftp ftp://ftp.up.pt/pub/ubuntu-releases/cdicons/kubuntu-iso.png
